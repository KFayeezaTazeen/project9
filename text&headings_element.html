TEXTS & HEADINGS

Text and Headings

Understanding Text Elements:
Text elements in HTML are used to create and structure the text content of a
webpage. These elements include
<p>, <h1>, <h2>, <h3>, <h4>,
<h5>, <h6>, <a>, <em>, <strong>, <small>, <mark>, <abbr>,
<q>, <cite>, <dfn>, <sub>, <sup>, and <time>.
The <p> element is used to create a paragraph of text, while the <h1> through
<h6> elements are used to create headings of different levels. The<a>
element is used to create a link to another webpage or resource.
The <em> element is used to emphasize text, while the <strong> element is
used to indicate text of importance. The <small> element is used to indicate
small print such as fine print.

Here is an example of text elements in an HTML document:
<h1>This is a heading</h1>
<p>This is a paragraph of text. &lt;a href="https://example.com">
<p>This text is <em>emphasized</em> and this text is
<strong>important</strong>.</p>

In this example, the <h1> element is used to create a heading, the <p>
element is used to create a paragraph of text, and the <a> element is used to
create a link. The <em> and <strong> elements are used to emphasize and
indicate importance of text.

It's important to note that text elements can contain other text elements, as well as
inline elements like <img> and <br> . Also, it is important to use the correct
text elements for the correct purpose to keep the HTML semantic and accessible.

Text elements refer to the various components or parts of a text, such as the title, headings,
paragraphs, sentences, and words. Understanding these elements is important for effectively
analyzing and interpreting a text. Additionally, text elements can also be used for organizing and
formatting a written work.

The <p> Element with an example
The <p> element is a HTML tag used to define a paragraph of text. The text contained within
the <p> tags will be displayed as a separate block of text, with a margin above and below it.

Example:

<p>This is a paragraph of text. It can contain multiple sentences and convey a main idea or
information.</p> <p>This is another paragraph of text. It is separate from the first one and can convey
different information or ideas.</p>

In the above example, two paragraphs are defined using the <p>element. The text contained
within each set of <p> tags will be displayed as separate blocks of text on the webpage.

In addition to defining a block of text, the <p> element can also have various attributes
added to it to change its appearance or behavior.

Example:

<p align="center" style="color:blue;">This is a paragraph of text that is centered and blue in color.</p>
In this example, the align attribute is used to center the text within the paragraph, and the style
attribute is used to change the text color to blue.
align attribute can have 4 values : left, center, right, justify
style attribute is used to define the CSS for the element.

Note: It's generally considered a better practice to use CSS stylesheets to define the styling for
elements, rather than using the style attribute.
The <br> and <hr> Elements with attributes
The <br> element is a HTML tag used to insert a line break in text, which means starting a
new line without starting a new paragraph. The <br> element does not have any closing tag
and it does not have any attributes.
Example:

<p>This is a paragraph of text.<br>This line of text will appear on a new line.</p>
In the above example, the <br> element is used to insert a line break between "text." and
"This line of text will appear on a new line."
The <hr> element is a HTML tag used to create a horizontal rule, which is a horizontal line
<hr>

In the above example, the <hr> element is used to create a horizontal line that can be used
as a visual separator between different sections of content on a webpage.
The <hr> element can also have various attributes added to it to change its appearance or
behavior.

Example:

<hr size="5" color="blue">

In the above example, the size attribute is used to change the thickness of the horizontal line and
color attribute is used to change the color of the horizontal line.

Note:

It's generally considered a better practice to use CSS stylesheets to define the styling for
<hr> element, rather than using the size, color attribute.
The <pre> Element with an example
The <pre> element is a HTML tag used to define preformatted text. The text contained within
the <pre> tags will be displayed exactly as it is written in the HTML code, preserving any
spaces, line breaks, and tabs.

Example:

<pre> This is an example of preformatted text. It preserves the spaces, line breaks, and tabs. This allows for
text to be displayed in a specific format, such as code. </pre>

In the above example, the text contained within the <pre> tags will be displayed exactly as it
is written in the HTML code, preserving the line breaks and spaces that are used to format the
text. This can be useful for displaying code snippets or other text that needs to maintain a
<p>As he said, <q>The best and most beautiful things in the world cannot be seen or even touched -
they must be felt with the heart.</q> </p>

In the above example, a short quotation is defined using the <q> element. The text contained
within the <q> tags will be displayed with quotation marks around it, indicating that it is a
quoted text.

It's also common to use the cite attribute to indicate the source of the quotation, for example:
<p>As he said, <q cite="https://www.brainyquote.com/quotes/helen_keller_102047">The best and most
beautiful things in the world cannot be seen or even touched - they must be felt with the heart.</q> </p>
In this case, the cite attribute is used to indicate the source of the quote (URL in this case).

Note: <q> element is useful to indicate a short inline quotation, it can be nested in other
elements such as <p> or <div>
The <abbr> Element with an example :
The <abbr> element is a HTML tag used to define an abbreviation or an acronym. The text
contained within the <abbr> tags will be displayed in a specific format, indicating that it is an
abbreviation or an acronym.
Example:

<p>The <abbr title="World Wide Web">WWW</abbr> was invented by Tim Berners-Lee</p>
In the above example, the <abbr> element is used to indicate that "WWW" is an abbreviation
for "World Wide Web". The title attribute is used to provide additional information about the
abbreviation. When users hover over the abbreviation, the full form will be displayed.

Additionally, some browsers may display the text in a specific format, such as a dotted underline,
to indicate that it is an abbreviation.

Note: <abbr> element is useful to indicate an abbreviation or acronym, it can be nested in
other elements such as <p> or <div>
The <time> Element with an example
The <time> element is a HTML tag used to define a date and/or time. The text contained
within the <time> tags will be displayed as a date and/or time, and can be used to provide
additional information about the date and/or time in a machine-readable format.

Example:

<p>The meeting is scheduled for <time datetime="2022-05-25T19:00">May 25th, 2022 at 7
PM</time></p>

In the above example, the <time> element is used to indicate a specific date and time for a
meeting. The datetime attribute is used to provide the date and time in a machine-readable format
(YYYY-MM-DDTHH:MM).
Additionally, the text displayed within the <time> tags will be the human-readable format of
the date and time, in this case "May 25th, 2022 at 7 PM"
This can be useful for search engines or other machines to understand the context of the date
and time and for assistive technologies to provide additional information about the date and time
to users.
Note: <time> element is useful to indicate a specific date and/or time, it can be nested in
other elements such as <p> or <div>
The <h1> to <h6> Elements with example
<h1> to <h6> elements are HTML tags used to define headings for a webpage.
These elements are used to indicate the different levels of importance of the text contained
within them.

The <h1> element is used for the most important heading, <h2> for the second most
important heading, and so on, with <h6> being used for the least important heading.

Example:

<h1>Main heading</h1> <h2>Secondary heading</h2> <h3>Tertiary heading</h3>
<h4>Quaternary heading</h4> <h5>Quinary heading</h5> <h6>Senary heading</h6>
In the above example, different levels of headings are defined using the <h1> to <h6>
elements. The text contained within each set of <h1> to <h6> tags will be displayed as a
separate block of text on the webpage with different levels of font size and weight.

These headings are used to organize the content of the webpage and to make it easier for users
to scan and understand the content of the webpage. They also help search engines to
understand the structure and the importance of the content on the webpage.

Note: These heading elements are used for SEO and accessibility, it's recommended to use them
in a hierarchical order and not to skip any heading level.
The <header> Element with an example
The <header> element is a HTML5 tag used to define a container for introductory content or
a set of navigational links. The content within the <header> element typically includes the
site logo, site title, site slogan or site header, and site navigation.

Example: <header> <h1>My Website</h1> <nav> <ul> <li><a
href="#">Home</a></li> <li><a href="#">About</a></li> <li><a

In the above example, a header is defined using the <header> element. The <header>
element contains the site title "My Website" and a navigation menu.
It's usually placed at the top of a webpage and it can be used once per page or document, and it
commonly contains the site branding and site navigation.

Note: <header> element is useful to group introductory content or set of navigational links, it
can be nested in other elements such as <body> or <article>.
The <nav> Element with an example
The <nav> element is a HTML5 tag used to define a section of a webpage that contains
navigation links. The links within the <nav> element are intended to allow the user to
navigate the website.
Example: <nav> <ul> <li><a href="#home">Home</a></li> <li><a
href="#about">About</a></li> <li><a href="#services">Services</a></li>
<li><a href="#contact">Contact</a></li> </ul> </nav>
In the above example, a navigation menu is defined using the <nav> element. The <nav>
element contains a list of links to different sections of the website, such as "Home", "About",
"Services", and "Contact".

It's commonly placed in the header or footer of a webpage and it can be used multiple times per
page or document, and it commonly contains the site navigation.

Note: <nav> element is useful to group a set of navigation links, it can be nested in other
elements such as <header> or <footer>
The <main> Element with an example
The <main> element is a HTML5 tag used to define the main content of a webpage. The
content within the <main> element should be unique to the document, excluding content
that is repeated across a set of documents such as site headers, footers, navigation, and
sidebars.
Example: 

        
<main>
<h1>Welcome to my website</h1>
<p>This is the main content of my website. Here you can find information about my services, my portfolio and how to contact me.</p>
<section>
<h2>About me</h2>
<p>I'm a web developer with 5 years of experience in the field.</p>
</section>
 <section> 
 <h2>My Services</h2> 
 <ul> 
 <li>Web Development</li>
 <li>E-commerce solutions</li> 
 <li>Search Engine Optimization</li> 
 </ul> 
 </section>
</main>

In the above example, the main content of the webpage is defined using the <main>
element. The <main> element contains the title "Welcome to my website" and the main
information about the website, such as a brief description of the website, information about the
developer and the services offered.
It's commonly placed in the body of the webpage, it can be used once per page or document and
it's meant to provide the main content of the webpage.

Note: <main> element is useful to group the main content of a webpage, it can be nested in
other elements such as <body> or <article>.

diseases.</p> <p>The research team behind the discovery, led by Dr. Jane Smith, will be presenting their
In the above example, an article is defined using the <article> element. The <article>
element contains a news story about a new discovery in the field of medicine. The story is
self-contained and makes sense on its own, it can be distributed or syndicated independently.
