Unordered Lists with an example

An unordered list in HTML is created using the <ul> element. The <ul> element is used
to define an unordered list, and each list item is defined using the <li> element. The items in
an unordered list are typically displayed with bullet points or dots.

Example: 
                                        
                                <ul> 
                                    <li>Apples</li> 
                                    <li>Bananas</li> 
                                    <li>Oranges</li> 
                                    <li>Pears</li>
                                </ul>

In the above example, an unordered list is defined using the <ul> element. The list items are
defined using the <li> elements, and the items will be displayed with bullet points or dots.
The items in this example are "Apples", "Bananas", "Oranges" and "Pears".
Note:

<ul> element is used to create an unordered list,
and <li> element is used to define each list item, it can be nested in other elements such as <body>
or <section>.

In the above example, the list-style-type property is used to change the bullet points of the
unordered list to squares. The font-size, color and margin-bottom properties are used to change the
font size, color, and the margin of the list items.

This way you can apply styles to specific lists while keeping the default styles for other lists.

Sorting Lists with an example:

Sorting lists in HTML can be done by using JavaScript or jQuery. JavaScript provides built-in
functions such as sort() and reverse() that can be used to sort list items.
Here is an example of using JavaScript to sort an unordered list alphabetically: Copy code 

    
<ul id="myList">
           <li>Item A</li>
           <li>Item B</li>
           <li>Item C</li>
</ul>


    
<script>
var list, i, switching, b, shouldSwitch; list = document.getElementById("myList");
switching = true; /* Make a loop that will continue until no switching has been done: */ while
(switching) {
// start by saying: no switching is done: switching = false;
b = list.getElementsByTagName("LI");
// Loop through all list items:
for (i = 0; i < (b.length - 1); i++) {
// start by saying there should be no switching:
shouldSwitch = false;
/* check if the next item should switch place with the current item: */
(b[i].innerHTML.toLowerCase() > b[i + 1].innerHTML.toLowerCase())
{
/* if next item is alphabetically lower than current item, mark as a switch and break the
loop: */ shouldSwitch = true; break;
} }
if (shouldSwitch) {
/* If a switch has been marked, make the switch and mark the switch as
done: */ b[i].parentNode.insertBefore(b[i + 1], b[i]); switching = true;
} }
</script>

In this example, the getElementById() method is used to select the unordered list with an id of
"myList". The getElementsByTagName() method is used to select all the list items within the
unordered list. The sort() function sorts the list items alphabetically using the innerHTML
property.
